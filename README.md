# SanyoniBMS

## Instruction
<b>유니티</b>로 만든 <b><a href="https://ko.wikipedia.org/wiki/Be-Music_Script">BMS</a></b>포맷을 채용한 <b>건반리듬게임</b>입니다.

[![Preview](https://img.youtube.com/vi/nigO9p7VPcw/0.jpg)](https://www.youtube.com/watch?v=nigO9p7VPcw) <br>

시연 영상(youtube): https://www.youtube.com/watch?v=nigO9p7VPcw <br>


본인은 Ez2AC시리즈와 DJMax시리즈 및 기타 리듬게임들의 오랜 열혈팬입니다!

이번 SanyoniBMS는 시스템적으로는 기존 BMS들의 완벽한 호환을, 디자인적으로는 DJMax Respect V를, 스킨 기어는 Ez2AC를 모방하려 했습니다.

## Control
기본적인 게임 플레이 가능 (곡 선택 화면→로딩 화면→게임 플레이 화면→결과 화면)

### In Select Music Scene
위아래 방향키를 통해 플레이할 곡 선택 가능

좌우 방향키를 통해 플레이할 패턴 선택 가능

Enter키를 통해 플레이 시작

### In Game Scene
1/2번 키를 통해 배속 조정 가능

F1키를 통해 Full Auto (완전 자동연주)

F2키를 통해 Scratch Auto (스크래치만 자동 연주)

턴테이블은 왼쪽 Shift, 그 외의 나머지 버튼들은 왼쪽부터 순서대로 S D F Space J K L로 입력할 수 있습니다.

## Supported Files
BMS: .bms, .bme, .bml

Audio:.wav, .ogg, .mp3

BGA: Image-Sequence(.bmp, .jpeg, .png)
